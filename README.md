# 图书管理系统 python813

## 项目概述

图书管理系统是一个集成了实时数据获取与管理的平台，旨在为图书馆管理员提供高效的书籍管理工具。该系统通过前后端分离的方式，使用 Java Spring Boot 和 Vue.js 技术栈实现前端界面，利用 Python Flask 框架获取实时书籍数据。

## 技术栈

- **前端：** Vue.js
- **后端：** Java Spring Boot
- **实时数据获取：** Python Flask

## 功能模块

- **实时数据同步：** 系统可以实时同步京东新书数据信息。
- **书籍管理：** 管理员可以实时同步上架最新书籍。

## 系统角色

- **图书馆管理员：** 负责管理图书信息，上架新书。

## 运行环境

- **后端：** 需要安装 Java 运行环境。
- **前端：** 需要现代浏览器支持。
- **实时数据获取服务：** 需要安装 Python 环境。

## 部署步骤

1. 部署后端服务。
2. 部署前端界面。
3. 部署实时数据获取服务。

## 目录结构

```
project/
├── backend/        # 后端代码目录
├── frontend/       # 前端代码目录
└── data_service/   # 数据获取服务代码目录
```

## 核心亮点

- **实时性：** 系统可以实时获取最新的书籍信息，提升图书馆信息更新速度。
- **扩展性：** 基于成熟的技术栈，方便后续功能的扩展和升级。
